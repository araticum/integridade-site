# Validacao Publica de Assinaturas

Este repositorio contem somente os artefatos publicos para validacao de autenticidade dos documentos do Programa de Integridade.

## Conteudo

- `assinaturas/manifesto-sha256.txt`
- `assinaturas/manifesto-sha256.txt.asc`
- `assinaturas/chave-publica.asc`
- `verify-docs.sh` para verificacao local por terceiros

## Verificacao

```bash
./verify-docs.sh
```

O repositorio de origem dos documentos permanece privado.
